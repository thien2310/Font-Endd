media query => responsive ( thiết kế đáp ứng )
        vd laptop có 3 cột 
        tab máy tính bảng 2 cột 
        mobile có 1 cột 
        https://www.w3schools.com/css/css3_mediaqueries.asp

        *** 
        screen là thiết bị media : computer , tab , mobile 
        min-width: 480px : biểu thức điều kiện hiểu đơn giản là áp dụng cho màn hình có độ rộng từ 480px trở lên 
        @media screen and (min-width: 480px) {
            body {
                background-color: lightgreen;
            }
            p {
                font-size: 14px;
            }
        }
