thuộc tính box-sizing 
        chiều rộng thực tế = width + border left + border right + padding left + padding right 
        chiều rộng = 150 + 6 + 6 + 20 + 20 = 202 
        chiều cao thực tế : height + border top + border bottom + padding top + padding bottom 
        chiều cao = 150 + 6 + 6 + 20 + 20 = 202 
        => khi viết css cho 1 thẻ việc thêm padding và border vào thay đổi width thực tế và height thực tế 
        => giải pháp dùng box-sizing
